# NoteTrack

**NoteTrack** is a collaborative note-taking and practice platform for students and teachers. It allows students to take notes, submit them to their teachers for review, and practice with teacher-created questions. Teachers can monitor student progress and provide targeted exercises.

---

## Features

### Student Features
- Create notes for each class.
- View previously submitted notes.
- Complete practice questions created by teachers.
- See which notes have been edited by teachers.

### Teacher Features
- Review and edit student notes.
- Create practice questions for specific students.
- Monitor student progress through submitted notes and practice performance.

---

## Tech Stack

- **Frontend:** React.js, React Router, Axios
- **Backend:** Node.js, Express.js
- **Database:** Firebase Firestore
- **Styling:** CSS
